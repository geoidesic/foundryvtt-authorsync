# Foundryvtt Authorsync

A FoundryVTT module

## Installation

1. Download the module from the [latest release](https://github.com/geoidesic/foundryvtt-authorsync/releases/latest)
2. Extract the zip file to your FoundryVTT `Data/modules` directory
3. Restart FoundryVTT
4. Enable the module in your world settings

## Development

This module is built using the TyphonJS framework with Svelte.

### Prerequisites

- Node.js 18+
- Bun (recommended) or npm

### Setup

```bash
bun install
bun run dev
```

### Building

```bash
bun run build
```
